1. ANÁLISIS

SUPERCLASE: Ejercicio7_Contacto  (abstracta)
  Atributos (protected, heredables):
    - nombre   : String
    - telefono : String

  Restricciones:
    - Dos contactos son iguales si tienen el mismo nombre (sin distinguir mayúsculas)

  Constructores:
    1. Contacto(nombre, telefono)

  Métodos:
    - equals(Object)  : boolean   compara por nombre
    - hashCode()      : int
    - getters / setters de nombre y telefono

SUBCLASE: Ejercicio7_Persona  extends  Ejercicio7_Contacto
  Atributos propios:
    - cumpleaños : String  (formato "dd/mm/aaaa")

  Constructores:
    1. Persona(nombre, telefono, cumpleaños)

  Métodos:
    - getters / setters de cumpleaños

SUBCLASE: Ejercicio7_Empresa  extends  Ejercicio7_Contacto
  Atributos propios:
    - paginaWeb : String

  Constructores:
    1. Empresa(nombre, telefono, paginaWeb)

  Métodos:
    - getters / setters de paginaWeb

CLASE: Ejercicio7_Agenda
  Atributos:
    - contactos : ArrayList<Contacto>

  Constructores:
    1. Agenda()   (agenda vacía)

  Métodos:
    - addContacto(Contacto)       : void      comprueba duplicados antes de añadir
    - borrarContacto(nombre)      : void
    - buscarContacto(nombre)      : Contacto  devuelve null si no existe
    - listarContactos()           : void

CLASE EJECUTABLE: Ejercicio7_Main
  - Crea una Agenda
  - Añade personas y empresas
  - Prueba búsqueda, borrado y listado
