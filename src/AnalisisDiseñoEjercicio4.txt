1. ANÁLISIS

SUPERCLASE: Ejercicio4_Vehiculo  (abstracta)
  Atributos (protected, heredables):
    - marca           : String
    - modelo          : String
    - color           : String
    - matricula       : String
    - motorEncendido  : boolean
    - marchaActual    : int
    - velocidadActual : int

  Valores por defecto:
    - motorEncendido  = false
    - marchaActual    = 0
    - velocidadActual = 0

  Constructores:
    1. Vehiculo(marca, modelo, color, matricula)

  Métodos:
    - arrancar()           : void   enciende el motor si estaba apagado
    - parar()              : void   para y apaga el motor
    - subirMarcha()        : void   sube una marcha (máximo 5ª)
    - bajarMarcha()        : void   baja una marcha (mínimo 1ª)
    - getters / setters de todos los atributos

  Rangos de marcha para la simulación:
    1ª :   0 - 30  km/h
    2ª :  30 - 50  km/h
    3ª :  50 - 70  km/h
    4ª :  70 - 100 km/h
    5ª : 100+      km/h

SUBCLASE: Ejercicio4_Turismo  extends  Ejercicio4_Vehiculo
  Atributos propios:
    - numPlazas : int     (por defecto 5)
    - tipoUso   : String  (por defecto "particular")

  Constructores:
    1. Turismo(marca, modelo, color, matricula)
         numPlazas y tipoUso por defecto
    2. Turismo(marca, modelo, color, matricula, numPlazas, tipoUso)

  Métodos:
    - getters / setters de numPlazas y tipoUso

SUBCLASE: Ejercicio4_Camion  extends  Ejercicio4_Vehiculo
  Atributos propios:
    - pesoMaxAutorizado  : double   (por defecto 1000.0 kg)
    - mercanciaPeligrosa : boolean  (por defecto false)

  Constructores:
    1. Camion(marca, modelo, color, matricula)
    2. Camion(marca, modelo, color, matricula, pesoMaxAutorizado, mercanciaPeligrosa)

  Métodos:
    - getters / setters de pesoMaxAutorizado y mercanciaPeligrosa

SUBCLASE: Ejercicio4_Motocicleta  extends  Ejercicio4_Vehiculo
  Atributos propios:
    - cilindrada : int  (por defecto 50 CC)

  Constructores:
    1. Motocicleta(marca, modelo, color, matricula)
    2. Motocicleta(marca, modelo, color, matricula, cilindrada)

  Métodos:
    - requiereCarnet() : boolean   devuelve true si cilindrada >= 125 CC
    - getters / setters de cilindrada

CLASE EJECUTABLE: Ejercicio4_Main
  - Crea un Turismo en reposo
  - Pide velocidad objetivo y tiempo de espera por teclado
  - Simula: arrancar → acelerar → mantener → desacelerar → parar
