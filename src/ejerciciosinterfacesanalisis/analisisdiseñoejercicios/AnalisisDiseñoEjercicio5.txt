  ANÁLISIS Y DISEÑO - EJERCICIO 5 - Biblioteca (Ficha)

1. ANÁLISIS

SUPERCLASE: Ejercicio5_Ficha  (abstracta)
  Atributos (protected, heredables):
    - id     : int
    - titulo : String

  Constructores:
    1. Ficha(id, titulo)

  Métodos:
    - prestar() : int   {abstract}   cada subclase devuelve sus días de préstamo
    - getters / setters de id y titulo

SUBCLASE: Ejercicio5_Libro  extends  Ejercicio5_Ficha
  Atributos propios:
    - autor     : String
    - editorial : String

  Constructores:
    1. Libro(id, titulo, autor, editorial)

  Métodos:
    - prestar() : int   devuelve 15
    - getters / setters de autor y editorial

SUBCLASE: Ejercicio5_Revista  extends  Ejercicio5_Ficha
  Atributos propios:
    - numero : int
    - año    : int

  Constructores:
    1. Revista(id, titulo, numero, año)

  Métodos:
    - prestar() : int   devuelve 10
    - getters / setters de numero y año

SUBCLASE: Ejercicio5_DVD  extends  Ejercicio5_Ficha
  Atributos propios:
    - director : String
    - año      : int
    - tipo     : String  ("película", "documental" o "serie")

  Constructores:
    1. DVD(id, titulo, director, año, tipo)

  Métodos:
    - prestar() : int   devuelve 5
    - getters / setters de director, año y tipo

CLASE EJECUTABLE: Ejercicio5_Main
  - Crea un Libro, una Revista y un DVD
  - Los recorre como array de Ficha (polimorfismo) e imprime los días de préstamo
